8 XPI features

8.1 Offset adjust

The offset adjustment is  is a global IPEmotion feature and very useful function to check and update

the physical measurements to the configured measurement range. With a offset operation you can

shift the current sensor signals on the analog inputs as your new base reference. The offset

function can be performed during the online measurements of the system.

It will be available if it is supported by the PlugIn (software) and the measurement channel

(hardware).

IPEmotion provide the extended Offset and target value adjustment with PlugIns IPETRONIK-X ≥
V02.00 and the following modules:

M-SENS2 / Mx-SENS2 4

M-SENS2 (with DSP, with 250 Hz Filter)

M-SENS (with DSP)

M-SENS 8 / M-SENS 8plus (with DSP)

M-SENS3 8

Mx-STG2 6 / Sx-STG

SIM-STG

The reference value can be defined by the user. As soon as the adjustment has finished, offset

values, available measurement ranges and results of the adjustment command are displayed.

STG devices supports a real adjustment by hardware. This provides the user a full measurement

range.

Error can occur if

Sx-STG: no sensor connected, Sensor mode is Strain gage and Sensor excitation is not 0V

Mx-STG2 6: no sensor connected (offset adjust is possible only with connected sensor/cable)
